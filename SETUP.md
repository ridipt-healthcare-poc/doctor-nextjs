# Doctor App - Setup Guide

## Quick Start

### 1. Create Environment File

Create `.env.local` in the doctor-app directory:

```
NEXT_PUBLIC_API_URL=http://localhost:8080
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Backend Server

```bash
cd ../healthcare-backend-v2
npm start
```

### 4. Start Next.js App

```bash
npm run dev
```

Visit: http://localhost:3000

## Default Credentials

Use existing doctor credentials from your backend database.
